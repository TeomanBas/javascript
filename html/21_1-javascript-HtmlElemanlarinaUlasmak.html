DOM(BELGE NESNE MODELİ),HTML ve CSS programlama dilleri arasında birstandartoluşturarak bu dillerin HTML VE CSS yi programala deyimleri ile etkin kullanmasına yardımcı olan bir nesne modelidir. bu nesne modeli sayfa yüklendiğinde oluşturulur.

ELEMANLARA ULAŞMAK

DOM İLE HTML elemanlarına ulaşmak için aşağıdaki kodlar kullanılabilir.

  1. HTML etiketlerinin "id" özelliğini kullaranarak document.getElementById(id)
  2. HTML etiket adı ile erişim document.getElementByTagName(name)
  3. HTML class özelliği ile erişim için document.getElementsByClassName(name)

ELEMANLARI DEĞİŞTİRMEK

  1. elemanın değerini değiştirmek için element.innerHTML=yeni değer.
  2. elemanın özelliği değiştirmek için element.attribute=yeni değer veya element.setAttribute(özellik adı ,yeni değeri)
  3. elemanın stilinin değiştirilmesi element.style.property=yeni değer

ELEMANLARI EKLEME/SİLME

  1. HTML elemanı oluşturmak için document.createElement(Eleman)
  2. HTML elemanını silmek için document.removeChild(eleman)
  3. HTML elemanını eklemek için document.appendChild(Elemen)
  4. HTML elemanı değiştirmek için document.replaceChild(eleman)
  5. HTML çıktı ekranına metin yazdırmak için document.write(Metin)
