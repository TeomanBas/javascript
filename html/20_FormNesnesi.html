FORM NESNESİ

formlar ziyaretçi ile etkişimde çok fazla kullanıldığı için javascript açısında html in en önemli nesneleridir.form oluşturmak için javascript kodlarına gerek yoktur ama form üzerindeki gerçekleşen işlemlerin yorumlanmasında javascript kulanılmaktadır.

javascript açısından formlar birer nesnedir.formun javascript için adı forms dur ve tüm formlar forms[] dizisi altında tutulur.Form üzeirndeki elemanlarda elements dir ve elements[] dizisi altında tutulur.bir forma ulaşmak için window.document özelliği ile birlikte forms dizisi veya formun adı kullanılabilir aynı pencerede işlem yapılırken window eklemeye gerek yoktur.

form içerisinde bulunan nesnelerin değerlerini okumak veya değiştirmek için nesnein value özelliği kullanılır.Text gibi bqazı form elemenlarının value değerleri değişken olabileceği gibi radio butonlarının value değerleri her zaman sabittir.

bir form nesnesinin value değeri aşağıdaki yötemle alınabilir 
var deger=document.formadi.nesneadi.value
bir form nesnesinin value değeri aşağıdak yöntemle değiştirilebilir.
document.formadi.nesneadi.value="yeni deger"
