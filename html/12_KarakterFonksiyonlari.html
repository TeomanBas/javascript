KARAKTER FONKSİYONLARI

javascript içerisinde değişken türü tanımlamaya gerek kalmadan değişken tuttuğu değere göre tür almaktadır.karakter türündeki değişkenlere değer atanırken değer tırnak içerisinde yazılmalıdır.

var deger="merhaba";
var deger="55";

yukarıdaki tanımlamalar değer türündeki tanımlamalardır. 55 de karakter türündedir çünkü tırnak içerisinde yazılmıştır.

var x ='ali'nin kalemi';//yanlış
var y ='ali/'nin kalemi';//doğru
var z ="ali'nin kalemi";//doğru

karakter türündeki ifadeleri birleştirirken "+" operatörü kullanılır.

var a ="emeğiniz";
var b ="oyuncağımızdır";
var c =a+b;//ab değişkeninin değeri 'emeğiniz oyuncağımızdır' olacaktır.

LENGTH

karakter türündeki bir değişkenin barındırdığı karakter sayısını verir.

var z="bu sene iyi para kazandık.";
var uzunluk_z=z.length// değişkenin değeri 26 olacaktır.

indexOf()

string değişken içerisinde yer alan bir karakter ya da karakter grubunun konumu hakkında bilgi verir. Sonuç -1 ise belirtilen karakter ya da karakter grubu yoktur. 0 veya daha büyükse dönen değer karakterin bulunduğu index değerini belirtir. Karakterlerin indeks değeri 0(sıfır)'dan başlar.boşluklarda karakter olarak sayılmaktadır. 

var a="gaziosmanpaşa üniversitesi";
var b=a.indexOf("G");//G değeri ilk karakter olduğu için 0 verecektir.
var b=a.indexOf("x");//x karakteri olmadığı için -1 verecektir.

chartAt()

parametre olarak verilen index değerine karakter kümesi içinde içerisinde karşılık gelen değeri verir.

var a="okulunda adı var ";
var b=a.chartAt(7);//0'dan başladığı için /.karakter d olacaktır.
