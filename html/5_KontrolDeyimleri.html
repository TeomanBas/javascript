KONTROL DEYİMLERİ

sayfa içerisinde belirli koşula bağlı işlemler yaptırmak birden fazla tekrar etmesi gereken işlemler veya istenilmeyen hataları önlemek için kontrol deyimleir ve döngüler kullanılır.koşula bağlı işlemler için if ve switch yapısı döngüler için for ve while yapısı kullanılır.

karşılaştırma deyimleri

sayfa içerisinde çoğu zaman çeşitli koşullara bağlı işlemler gerçekleştirilir.Mesela web tarayıcısının türüne göre o anki saate göre veya kullanıcıdan alınan değere göre işlemler gerçekştirilmek istendiğinde karşılaştırma deyimleri kullanılır.Koşula bağlı deyimler yazılırken koşul ifadesi için karşılaştırma operatörleri birden fazla koşul için mantıksal operatörler kullanılmaktadır.Javascriptte kullanılan karşılaştırma deyimleri karmaşık scriptler geliştirmeye yardımcı olur.

if - else

çoğu programlamadilinde oluduğu gibi javascript içerisinde de koşula bağlı işlemlerde if-else yapısı çoğunlukla kullanılmaktadır.Belirtilen koşula bağlı tek satırlık komut çalıştırılacaksa if ve else den sonra {} parantezleri kulanmaya gerek yoktur.birden fazla koşula bağlı işlemlerde matıksal operatörler kullanılması ve koşulların parantez içerisinde olması gerekir.else yapısının kullanım zorunluluğu yoktur.

aşağıda if-else yapısını üç farklı kullanımı verilmiştir.

1. kullanım

sadece koşulun sağlanması durumuna göre işlemler gerçekleştirir

if(koşul_ifadesi)
{
koşul sağlanırsa gerçekleşecek
}
2. kullanım

koşulun sağlanması veya sağlanmaması durumuna göre işlemler gerçekleştirir.

if(koşul_ifadesi)
{
koşul sağnırsa gerçekleşecek işlemler
}
Else{
koşul ifadesi sağlanmassa gerçekleştirilebilecek işlemler
}
3. kullanım

farklı koşullara göre farklı işlemler gerçekleştirmek istenildiğinde kullanılır.

if(koşul_ifadesi_1)
{
koşul 1 için gerçekleşecek işlemler
else if(koşul_ifadesi_2)
{
koşul 1 sağlanmassa burası çalıştırılır 
}
else if(koşul_ifadesi_3)
{
koşul2 sağlamassa buradaki işlemler gerçekleşecek.
}
.
.
.
else
{
tüm if koşulları sağlanmadığında en son gerçekleştirilecek olan işlemler burada yer alır.
}

SWİTCH-CASE

seçilen değişkenein belirlenen değerlerden hangisine uygun olduğuna bakarak o değer için belirlenmiş işlemleri gerçekleştirir.Her case ifadesinde yaptırılacak işlemlerden sonra break komutuyla switch yapısından çıkılması gerekmektedir.Aksi takdirde koşula bakılmaksızın diğer komutlarda çalıştırılır.Belirtilen değerlerin hiçbiri sağlanmıyorda default için komutlar işleme konulur.Case ifadesinde belirtilen bir değer olabilecğei igibi bir koşul ifadeside olabilir. belirtilen değer sayısal türde ise tırnak içerisine almaya gerek yoktur fakat değer karakter kümesi türündeyse eblirilecek değer tırnak içerisinde olmalıdır.

kullanım şekli:
switch (değişken)
{
case d1:
	değişkenimizin değeri d1 ise gerçekleştirilecek işlemler
}break;
{
case d2:
	değişkenimizin değeri d2 ise gerçekleştirilecek işlemler
}break;
case d3:
	değişkenimizin değeri d3 ise gerçekleştirilecek işlemler
}break;
. 
. 
.
default:
	değişkenimiz değeri d1,d2,d3 değilse gerçekleştirilecek işlemler
	break;
	}

aynı işlem birden fazla koşula bağlıysa koşulları ayrı ayrı yazarak he koşul için aynı işlemi yazabilirsiniz ama bu programcılık açısında pek doğru bir yol değildir.Bunun yerine iki case ifades alt alta yazılarak birinci case ifadensinden sonra break kullanılmaz.

örnek olarak:

switch (değişken)
{
case d1:
case d2:
değikenimizin değeri d1 veya d2 ise gerçekleştirilecek işlemler
}break;
