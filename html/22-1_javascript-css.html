css(cascading style sheets) html olarak hazırlanmış sayfayı dinamik hale getirmenin ve görsel olarak süslemenin en önemli unsurlarında birisidir.Örneğin sayfa kaydırma çubuklarının rengini değiştirme nesneye yazılan bilginin otomatik büyük harfe çevrilmesi linklere kısa yoldan efekt verme gibi birçok işlem css kullanarak gerçekleştirilebilir.Javascript yardımıyla css özellikleri değiştirilebilir veya yeni özellik verilebilir.

yukarıdaki örnekte this.bgColor yöntmeiyle zemin rengi this.style.color yöntemiyle yazı rengi değiştirilmektedir.ONmouseout olayında bu fonksiyonlara renk verilmeden boş bırakılmasının nedeni default yani eski haline dönmesi içindir bu ili yöntem sadece tablolar için geçerli değildir istenilen tüm html nesneleirinde kullanılabilir.

javascript yardımıyla css kullanırken this eki her zaman geçerli değildir this eki her zaman geçerli değildir this eki kodun yazıldığı nesne üzeirnde değişiklik yapmak için kullanılır farklı bir nesnein özelliğini değiştirmek için aşağıdaki yöntemlerden birisi kullanılmaklıdır.

document.all.id.style.değişecek_özellik="yeni deger"; document.getElementById(id).style.değişecek_özellik="yeni deger";

id:sayfanızda bulunan ve özelliği değiştirilecek nesnein id değeridir.nesne oluşturulurken istenilen id için istenilen bir değer verilebilir(< table id="tablom" >)

değişecek_özellik:Nesnenizin değiştirmek istediğiniz özelliğidir.Bu özellik rasgele bir bilgi değildir.Bu özelliklerin bir kısmı aşağıda tablo halinde verilmiştir.
