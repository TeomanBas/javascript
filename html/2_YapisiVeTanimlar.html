DOM (DOCUMENT OBJECT MODEL)

DOM html ve css ile programlama dilleri arasında bir standart oluşturur ve dilleri programlama deyimleri ile etkin kulanmasına yardımcı olur.Yani dom html ve css nesneler bütününe çeviren bir standarttır.dom ile java script bir web sayfasının tüm html ve css kodlarına erişir ve bunları değiştirebilir. bir web sayfası yüklendiğinde dom (web sayfasını modeli) oluşturulur.java script ile html css kodlarına müdahale edilebildiği için html ve css e programlama kabiliyeti kazandırmış oluruz.
